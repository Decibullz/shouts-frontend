# Shouts!

A fully featured social media app, that allows all users to interact with each other just by signing up!

## Tech Used

- HTML
- CSS3
- React.js
- Node.JS
- Express.JS
- Google Firebase
- Material UI
- Dayjs
- Redux
- Axios
- Postman
